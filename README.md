Projeto desenvolvido com Java Spring Boot para gerenciamento de passageiros

## Tecnologias

- Java
- Spring Boot
- Spring Data JPA
- H2 Database
- Swagger/OpenAPI
- Gradle

## Funcionalidades

- CRUD completo
- Validação de dados
- Tratamento de erros
- Documentação com Swagger
- DTOs para entrada e saída de dados  

## Documentação da API

Swagger disponível em:
