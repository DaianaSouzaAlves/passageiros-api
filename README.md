Projeto desenvolvido com Java Spring Boot para gerenciamento de passageiros

## Tecnologias

- Java
- Spring Boot
- Spring Data JPA
- H2 Database
- Swagger/OpenAPI
- Gradle

## Funcionalidades

- CRUD completo
- Validação de dados
- Tratamento de erros
- Documentação com Swagger
- DTOs para entrada e saída de dados
- Organização em camadas  

## Como executar

1. Clonar o projeto
2. Abra o peojeto na IDE
3. Executar aplicação
4. Acessar Swagger

## Swagger

http://localhost:8080/swagger-ui/index.html
